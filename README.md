## 🚀 Installation and Usage

To get started with this project, follow these steps.

1.  **Set up the virtual environment** (only the first time):
    ```bash
    make setup
    ```

2.  **Activate the environment**:
    ```bash
    source .venv/bin/activate
    ```

3.  **Install dependencies**:
    ```bash
    make install
    ```

4.  **Run the main code**:
    You can run the main script with:
    ```bash
    make run
    ```
    You can also open a Jupyter notebook for exploratory analysis:
    ```bash
    jupyter notebook notebooks/1.0-initial-exploration.ipynb
    ```
